Connect your own Google account to save your credentials privately in your Google Drive. Nobody else can access your data.

From Google Cloud Console → Credentials → OAuth 2.0 Client IDs

⚠ Each person using this app needs their own free Google API credentials. It takes about 5 minutes.
1
Create a Google Cloud project
Go to console.cloud.google.com → click New Project → name it "Login Vault" → Create.
2
Enable Google Drive API
APIs & Services → Library → search Google Drive API → Enable.
3
Create an API Key
Credentials → Create Credentials → API Key. Copy and paste it above.
4
Create OAuth 2.0 Client ID
Create Credentials → OAuth Client ID → Web application. Under Authorized JavaScript origins add: https://shahzaibali0308.github.io. Copy the Client ID and paste it above.
5
Set up OAuth Consent Screen
APIs & Services → OAuth Consent Screen → External → fill App name → under Test users add your Gmail → Save.
